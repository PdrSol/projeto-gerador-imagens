O projeto GerIAção tem como objetivo democratizar o acesso à criação visual, permitindo que qualquer pessoa possa gerar imagens incríveis a partir de descrições simples.

Utilizando o poder da Inteligência Artificial, a plataforma facilita o processo criativo e estimula a imaginação.

Tecnologias Utilizadas:

  • HTML5 para estruturação do conteúdo
  • CSS3 para estilização (em breve)
  • JavaScript (futuramente)
  • Modelos de IA para geração de imagens
